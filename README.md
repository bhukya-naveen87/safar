### Safar Project App:
#### Commands to Create:
- ```npm i -g expo-cli```
- ```expo init <project_name>```
- ```cd <project_name>```
- ##### open phone and scan QR code to run on your phone.
    - ```npm start```
- ##### To run on web:
    - ```npx expo install react-dom react-native-web @expo/metro-runtime```
    - ```npm run web```
